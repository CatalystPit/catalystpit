# CatalystPit

Live market intelligence platform. Every catalyst. Before the  bell.
 
## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your environment variables:
```bash
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY
```

3. Run locally:
```bash
npm run dev
```

4. Deploy to Vercel:
- Push to GitHub
- Import in Vercel
- Add ANTHROPIC_API_KEY in Environment Variables
- Deploy

## Stack
- Next.js 14
- React 18
- Anthropic Claude API (web search tool)
- TradingView Lightweight Charts
